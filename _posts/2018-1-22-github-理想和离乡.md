# 近况

去年熬夜把身体熬坏了，体重破纪录回到了150斤，最近忙着健身与养生，又重回140斤

# 体制

在学校和同学交流，发现大龄女青年想进体制，的确，进来结婚生子照顾家庭都不耽误，能给很多女同学想要的安全感。

# 心态

体制内呢，不需要你有能力，也不需要你有良好的社会交际能力，最重要的是心态，温水煮青蛙只是个人太要强，整个制度给不了你施展空间而已，但你一定要记好进来是为了什么，有的人图仕途，有的人图安逸，有的人图照顾家庭，在里面，别忘了自己的初衷，而能保得个人家庭的幸福即可。

# 2017

最大收获就是注册电气工程师考过了，还是很有含金量的证书，然后在新郑试点运营个小项目，感觉医院也不太适合自己，计算机真的是门很坑的学科，去哪里都感觉很鸡肋，但去公司又干不长久，看来除了学校研究院和单干或者继续去上级单位没啥好职业规划了 

# 2018

关键三年的第二年，除了司法考试外没什么特别强烈的需求，都劝我该结婚了，连堂弟都等着我结婚好，想想老家就剩我俩了（哭笑不得），我这个人很奇怪，要不都要，要不都舍弃，2016年我那么卑躬屈膝只要你们说的什么都接受，结果还是伤害我，那么我就只找最合适的，掺不了半点假。这里也希望堂弟获得独立和自信，不依赖他人给于能量而自己顶天立地

# 想要的生活

我本来也是很贪图安逸稳定的，每天和二三好友看看电影聊聊天，有个team一起健身，周末或者宅着或者做个副业，偶尔养养生(最近艾灸后学中医银针扎穴位)，我的成就感来自己与健硕的身体，良好的气质，丰富的内涵和一部分挣钱的能力。

但是真的不喜欢自己被当工具当枪使，工作中很多人感觉心理都有问题，我健身说我变态，我考证说我不务正业，耽误他们工作就威胁吓唬我，为什么都不想想是不是分配不均的原因呢？这个世界吗，讲究等价交换，舍弃了一些才能得到另一些，我不和你争小利而图大我，为什么要喋喋不休？算了，也许是处境不同吧，但真的没必要为不同的世界观论个长短深浅。

我还不能稳定下来，不说和那些在地方两年直接去了部委的大学同学比，也不论那些技术移民的bad boy，我想要的工作呢，不一定一定要天下名扬，但一定要有希望，我长久呆的地方，一定不要是死水，曾经过来人告诉我，能搞好自己副业，能考证，能独善其身，这个体制就很适合自己，但现在我感觉我不想多交流都不行，保持自我的思想独立都很难，当然，为了一些东西我愿意在不舍弃一些人的情况下等价交换，至于结果，就用时间来验证吧。
